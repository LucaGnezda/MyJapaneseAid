Usage Tips

  • MyJapaneseAid recommends installing Japanese kana input on your device, it will make it so much easier to work with, and learn kana.
  • When typing romaji specifically, MyJapaneseAid will automatically convert '-' after a vowel, into the equivalent long sound vowel.
  • Try using highlighters in the edit panel of language items. When applied correctly this will underline text across kana, romaji and english meaning when not in edit mode.
  • Highlighters are comma separated lists of character positions, with the first character position being 1. You can even create multiple highlighter sets with the '\' delimiter. When using sets, each comma position will highlight the same position in all sets. This is especially useful when you have both Kanji and Hiragana version in the same line. You can only include a position index once per highlighter, but this still gives some pretty awesome linked highlighting effects. for example the following text:
    • kana = わたしは[A]です (私は[A]です)
    • romaji = watashi wa [A] desu
    • meaning = I am [A]
  • Might be highlighted as follows:
    • kana = 1-3,4,5-7,8-9\12,13,14-16,17-18
    • romaji = 1-7,9-10,12-14,16-19
    • meaning = 1,,6-8,3-4
  • Language searching will allow both 'and' and 'or' searches if you include more than one term separated by a space.
  • The kana tab will present bot hhiragana and katakana in a variety of layouts. Extended katakana is also included (there is no equivalent hiragana for these sounds). Word of caution, this list might not be comprehensive as the set seems to evolve over time as the Japanese language adopts new loanwords.
  • I hope you enjoy using MyJapaneseAid.

Welcome...
MyJapaneseAid is a simple tool to help you build a personal repository of Japanese words, phrases and sentences you learn. It uses your local browser cache as a datastore, meaning no data is ever transmitted over the internet.
This is a personal project. You are welcome to use it, but I cannot commit to any form of support or bugfixing if you run into issues.
